## 🚀 Deploy in 3 Steps

### **1. Commit & Push**
```bash
cd /Users/mac/kaalay
git add .
git commit -m "feat: Integrate Suqafuran Express delivery features"
git push origin main
```

### **2. Set Environment Variables in Vercel**

Go to: **https://vercel.com**

Project: **frontend-v2** → Settings → Environment Variables

Add:
```
NEXT_PUBLIC_API_URL=https://api.kaalay.com
NEXT_PUBLIC_DELIVERY_API=https://app.suqafuran.com
NEXT_PUBLIC_DELIVERY_WS=wss://app.suqafuran.com
```

Save → Deployments → Redeploy

### **3. Test**
Visit: **https://kaalay.vercel.app/driver/delivery**
