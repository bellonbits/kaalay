# Kaalay Haley (Kuja Hapa)

## Overview
A ride-hailing and logistics platform optimized for Africa.

## Core Innovation
Precise location using what3words instead of unreliable addresses.

## Platforms
- Web (React + Ant Design)
- Mobile (Capacitor)
- Backend (NestJS)

## Roles
- Rider
- Driver
- Admin

## Constraint
No in-app payments. All payments happen offline.
